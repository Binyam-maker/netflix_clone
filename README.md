# Netflix Clone

This is a Next.js Netflix clone project.  I used [TMDB](https://www.themoviedb.org/) API to get data. 



It is built with

 - Next.js
 - Tailwind.css
 - Framer Motion
 - Next-Auth
 - Redux toolkit
 - Mongoose 
 -  bcrypt 

Live site can be seen here [`netflix_clone`](netflix-clone-sable-one.vercel.app).
